# Sports Hall Booking System

## 📖 Overview
The Sports Hall Booking System is developed on the ServiceNow platform to simplify the process of booking sports facilities. It allows users to reserve sports halls online, while administrators can manage bookings, schedules, and availability efficiently.

## 🎯 Objectives
- Automate the sports hall booking process.
- Reduce manual scheduling conflicts.
- Provide an easy-to-use booking interface.
- Improve booking management and reporting.

## ✨ Features
- User registration and login
- Sports hall booking
- View booking status
- Booking approval workflow
- Booking cancellation
- Admin dashboard for managing bookings

## 🛠️ Technologies Used
- ServiceNow Platform
- Flow Designer
- Service Catalog
- Forms and Tables
- Business Rules
- Notifications

## 📂 Project Structure
- Application Scope
- Tables
- Forms
- Workflows
- Business Rules
- Flow Designer
- Reports

## 🚀 How to Use
1. Log in to the ServiceNow instance.
2. Open the Sports Hall Booking application.
3. Select a sports hall.
4. Choose the date and time.
5. Submit the booking request.
6. Wait for approval and confirmation.
